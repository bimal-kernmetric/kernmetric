03 / Navigation Header

Purpose & Behaviors

Provides consistent global navigation context across all devices.
• Desktop: Responsive editorial layouts, direct CTA triggers.
• Tablet/Mobile: Toggles an absolute slide-down menu drawer, replacing sidebar options.

Accessibility Specs

• Toggles use explicit aria-expanded and aria-label tags.
• Interactive anchors support matching `24px` target shapes on touch devices.
